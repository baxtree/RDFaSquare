RDFa² is an on-line service which helps carrying out multi-topic annotation with RDFa. Under the guidance of RDFa², you can personalize the generated Web page by revising the XHTML+RDFa source code and get a preview on the fly. Alternatively, you can revise the intermediate template and then save it to your local machine for future reuse.

When adding data to a target entity (e.g., person, place or organization) on a Web page, a user is normally focused on the properties associated with this target and temporarily ignores other resouces on this page. Based on this, RDFa² provides a Topic Node Discovery approach recommending targets to you. This can be particularly helpful if you are not sure which entities are present inside an RDF document. Please use the button.

A single RDF document may have more than one resource in which users are interested. RDFa² allows you to annotate multiple topics derived from a single RDF document. Please use the button. On the other hand, sometimes you may want to annotate a Web page using resources derived from multiple contexts (RDF documents). RDFa² also allows you to carry out this task using the button.

By selecting the option , you can easily get a ready-to-publish XHTML+RDFa page with the full features of a normal Web page. But if you have already got your Web page as well as relevant RDF document(s) and want to reuse this page, you can select the option to get a code snippet which can be easily copied and pasted to your Web page's <body> section.

Finally, you may be a vocabulary curator or ontologist and want to publish your vocabulary in a more readable format. This is exactly where the option comes into play. In this scenario, the selection of topic nodes can be ignored.
