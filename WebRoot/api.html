This is an instruction about how to use API. A simple way of generating an XHTML+RDFa from a specific RDF file is to drag RDFa²'s lazy annotator to your bookmark bar. Thereafter, when you type in the URI into the address bar of your favorite browser, there is just one click away from the XHTML+RDFa snippet in terms of the automatically detected topical URI.

POST Query URL:
http://demos.inf.ed.ac.uk:8836/rdfasquare/baxtree/apis/FederatedTransformer
Parameters:
rdfurls : URLs of the RDF context files. Multiple URLs are seperated by ",".
topicuris : topic URIs that preexist in these RDF context files. Multiple URIs are seperated by ",".
Example: TBW ...

POST Query URL:
http://demos.inf.ed.ac.uk:8836/rdfasquare/baxtree/apis/SmartTransformer
Parameter:
rdfurl : The URL of the RDF context file.
Example: TBW ...

POST Query URL:
http://demos.inf.ed.ac.uk:8836/rdfasquare/baxtree/apis/TemplateCreator
Parameter:
rdfurl : The URL of the RDF context file.
topicuri : The topic URI that preexists in these RDF context file.
Example: TBW ...
