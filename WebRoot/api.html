API

This is an instruction about how to use API. A simple way of generating an XHTML+RDFa from a specific RDF file is to drag RDFa2's lazy annotator to your bookmark bar. Thereafter, when you type in the URI into the address bar of your favorite browser, there is just one click away from the XHTML+RDFa snippet for the automatically detected topical URI.

POST Query URL:
http://southrow.inf.ed.ac.uk:8838/rdfasquare/baxtree/apis/FederatedTransformer
Parameters:
rdfurls : URLs of the RDF context files. Multiple URLs are seperated by ",".
topicuris : topic URIs that preexist in these RDF context files. Multiple URIs are seperated by ",".
Example:

POST Query URL:
http://southrow.inf.ed.ac.uk:8838/rdfasquare/baxtree/apis/SmartTransformer
Parameter:
rdfurl : The URL of the RDF context file.
